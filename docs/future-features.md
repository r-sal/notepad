# Future Features

## Planned

- **Password-protected notes** — Lock individual notes behind a password so they require authentication before viewing or editing
- **Export / download note** — Allow users to export a note as a file (e.g., PDF, Markdown, HTML, plain text)

## Ideas

- **AI chat assistant** — An integrated AI chat that has full context of all your notes and can answer questions, summarize content, find connections between notes, etc.
